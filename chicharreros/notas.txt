-- Scroll por link vs navegación por link:
   El ul dentro del nav tiene un id que se usa en una función jquery para hacer el scroll
   personalizado al pinchar un link (dentro de la misma página), dicha función impide 
   navegar a otros html.
   Solución: quitar id="nav" del  <ul> o poner dos navegadores, uno para páginas html y otro
   para scrolling a secciones de la página actual.
